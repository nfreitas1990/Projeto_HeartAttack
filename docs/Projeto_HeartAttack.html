Contextualizao

O infarto agudo do miocrdio, conhecido como ataque cardaco (no ingls Heart Attack), est entre uma das doenas cardiovasculares mais mortais. Geralmente ocorre quando a circulao ou o fluxo sanguneo para o corao interrompido, fazendo com que o corao no receba oxignio. Leva apenas de seis a oito minutos sem oxignio para que o msculo cardaco pare de funcionar, levando o indivduo a morte.

No Brasil, em 2021, estima-se que 230 mil pessoas morreram por doenas cardiovasculares, e destes, 73.035 mil decorrentes de infarto (CNN,2021).


E se pudssemos predizer as pessoas que esto mais propensas a sofrer ataques cardacos?


Neste contexto, este estudo surge com o objetivo de desenvolver um sistema de predio para identificar, com base em caractersticas biolgicas,
indivduos com maior propenso a sofrer ataque cardaco. Alm de ajudar no diagnstico, possibilita melhor compreenso do problema ao mapear as caractersticas que mais se associam ao risco de ataque cardaco.


Cdigo para as anlises pode ser acessado GitHub nfreitas1990. Sugiro o acompanhamento das anlises com o cdigo aberto em Script_HeartAttack.R ou em .Rmd Projeto_HeartAttack.Rmd. Pois alguns cdigos foram ocultados para melhorar a visualizao.

Colinearidade

As variveis numricas foram testadas para evitar colinearidade. O mtodo de Spearman foi escolhido em detrimento do mtodo de Pearson para evitar assumir pressuposto com relao a normalidade dos dados.

  ha_numeric <- ha  |>
                    select(where(is.numeric))
  corrplot.mixed(cor(ha_numeric, method = "spearman"),lower = "number", upper = 'color')
A anlise indica que existe baixa correlao (<0.45) entre as variveis. Indicando que todas podem ser utilizadas para as anlises futuras, sem incorrer em risco de colinearidade.

Criando funes que sero usadas nesta seo

# Funo grafico_proporcao ( ) :para Histograma das propores
  grafico_proporcao <- function(coluna, bins=NULL, breaks = NULL, eixox = NULL){
    
    ha |> 
      ggplot(aes(y = as.numeric(output)-1, x = coluna)) +
      stat_summary_bin(size = 1, alpha = 0.1, colour = "white", bins = bins,breaks = breaks,
                       geom = "bar", fill = "royalblue", fun = function(x) 1,
                       na.rm = T) +
      stat_summary_bin(size = 1,alpha = 0.3, colour = "white", bins = bins,
                       breaks = breaks,geom = "bar", fill = "orange", na.rm = T) +
      stat_summary_bin(size = 2, alpha = 1, colour = "purple", bins = bins,
                       breaks = breaks, geom = "point", na.rm = T) + 
      stat_smooth(method = "glm", method.args = list(family = "binomial"),
                  se = FALSE, na.rm = T) +
      xlab(eixox)+
      ylab("Proporo Doena cardaca")+
      geom_point() +
      meu_tema}
    
    # Funo tabela_proporcao ( ): para tabela das propores
  
  tabela_proporcao <- function(coluna, breaks = NULL ){
    ha |> 
      mutate(
        coluna_faixa = cut(coluna, breaks = breaks)) |> 
      group_by(coluna_faixa) |> 
      summarise(
        n = n(),
        coluna = mean(coluna),
        p_output = mean(output == 1),
        logit_chance_output = log(p_output/(1-p_output)))}
